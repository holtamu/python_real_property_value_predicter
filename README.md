# 🏠 우리 집 근처 부동산 가격 예측 프로젝트

이 프로젝트는 텐서플로와 재미나이를 활용하여 실거래가를 예측하고 분석합니다.

프로젝트 주요 단계 요약

1. 환경 설정 및 데이터 수집: 라이브러리 설치 및 공공데이터 API 연동

2. 데이터 전처리 (EDA): 결측치 제거 및 학습에 적합한 형태로 데이터 가공

3. 텐서플로 모델링: 집값 예측을 위한 신경망 모델 구축 및 학습

4. Gemini 연동: 모델의 예측 결과에 대한 AI의 자연어 해석 및 분석 추가

5. 시각화 및 정리: 그래프를 통한 결과 확인 및 코드 모듈화
